#### Projeto IFoodles

#### Dependências

* prisma
* axios
* icons

* prisma install
```
npm install prisma@6.2.1
```

* prisma client
```
npm install @prisma/client@6.2.1
```

* iniciando o prisma
```
npx prisma init
```

* após criar os models rodar o comando prisma format
```
npx prisma format
```

* rodando migration
npx prisma migrate dev
